== Description ==
This is WP Chargebee, a WordPress plugin which allows you to seamlessly process payments and track your customers on your own WordPress website. 
Note that this plugin requires the Gravity Forms plugin to be installed and active on your WordPress website. Otherwise, WPChargebee will not activate on your site. See https://www.gravityforms.com/ for information about pricing and installation.
=Features=
WP Chargebee offers a wide range of features to help you track and process payments for your Chargebee plans, on your very own WordPress site frontend.

Display your Chargebee plans easily with the integrated plans page and shortcode
Allow your site's visitors to sign up to your Chargebee site as customers
Handle all card payments easily using an integrated Stripe Checkout integration
== Installation ==
1. Download the zip file from https://wordpress.org/plugins/wpchargebee/
1. Unzip it in your plugins directory (/wp-content/plugins/)
== Frequently Asked Questions ==
How do I use the plugin?
Step 1) Install and activate WP Chargebee
Step 2) Create a plan, form and payment/confirmation page
Step 3) Go to settings and fill in all the required fields for setup
Step 4) Use the available shortcodes on your WordPress pages
Step 5) Set any custom CSS on the plans as desired
